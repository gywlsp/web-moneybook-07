# web-moneybook-07

3,4주차 가계부 서비스 미션 7팀 박효진, 최연정

## Getting Started

### Clone Repository

```
git clone https://github.com/woowa-techcamp-2022/web-moneybook-07.git
```

### install Packages

```
npm install
```

### create env file

```
// .env
MYSQL_HOST=""
MYSQL_USER=""
MYSQL_PW=""
MYSQL_DB=""
```

### run application

```
npm run dev
```
